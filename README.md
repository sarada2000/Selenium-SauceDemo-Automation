# SauceDemo E2E Automation Suite

![Java](https://img.shields.io/badge/Language-Java-red) ![Selenium](https://img.shields.io/badge/Automation-Selenium-blue) ![TestNG](https://img.shields.io/badge/Framework-TestNG-green) ![Maven](https://img.shields.io/badge/Build-Maven-blueviolet)

This repository contains a **professional End-to-End (E2E) automated test suite** for the [SauceDemo](https://www.saucedemo.com/) e-commerce platform. The project demonstrates advanced **Selenium WebDriver** practices, including browser configuration, element interaction, and automated verification.

---

## üöÄ Key Features

- **Full Purchase Journey**: Automates the entire workflow from login to final checkout completion.  
- **Automated Assertions**: Uses **TestNG assertions** to programmatically verify success messages, e.g., `"Thank you for your order!"`.  
- **Security Bypass**: Integrates **ChromeOptions** to handle browser-level security popups (like password breach notifications) for smoother execution.  
- **Clean Architecture**: Structured with **Maven** and **TestNG** for professional dependency management and detailed reporting.
