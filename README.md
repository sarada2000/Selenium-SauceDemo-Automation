 SauceDemo E2E Automation Suite
This repository contains a professional End-to-End (E2E) automated test suite for the SauceDemo e-commerce platform. The project demonstrates advanced Selenium WebDriver practices, including browser configuration, element interaction, and automated verification.

Key Features
Full Purchase Journey: Automates the entire process from login to final checkout completion.

Automated Assertions: Implemented TestNG assertions to programmatically verify the success message ("Thank you for your order!").

Security Bypass: Integrated ChromeOptions to handle browser-level security popups (like the password breach notification) for smoother execution.

Clean Architecture: Structured using Maven and TestNG for professional dependency management and reporting.

Technology Stack
Language: Java

Automation: Selenium WebDriver

Testing Framework: TestNG

Build Tool: Maven

Driver Management: WebDriverManager
